Detailed Description

Main inventory management class that implements IInventoryQueue for InventoryItem.

Manages a queue of inventory items, updates HUD slots, and handles item instantiation in the player's hand. Provides methods to add items, cycle through them, and delegate queue operations. Inherits from MonoBehaviour for Unity integration.
