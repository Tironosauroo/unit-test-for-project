Detailed Description

This file contains the PlayerLook class, which handles camera rotation based on mouse input in the Unity game. It uses Unity's Input System to control player look direction and clamps vertical rotation.

The PlayerLook component manages mouse sensitivity, input reading, and applies rotations to the camera and player transform. It ensures smooth look controls with clamped pitch to prevent over-rotation.
