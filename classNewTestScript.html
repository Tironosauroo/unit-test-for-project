Detailed Description

Test class for InventoryQueue functionality.

This class provides comprehensive unit tests to validate the InventoryQueue implementation. It tests enqueue, dequeue, capacity resizing, reference clearing, and mixed operations. Each test method is designed to be independent and focuses on a specific aspect of the queue's behavior.
