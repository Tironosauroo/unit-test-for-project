Detailed Description

Test class for PlayerMovement functionality.

This class provides unit tests to validate the PlayerMovement's methods for handling movement and crouching. It includes setup and teardown to create and destroy test objects, ensuring clean test environments. Tests focus on input-driven movement, camera positioning, and method invocations via reflection.
